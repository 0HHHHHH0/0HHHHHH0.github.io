训练您自己的模型——长牡蛎

本指南提供了一个详细、逐步的教程，用于在长牡蛎数据上训练一个 OpenSpliceAI 模型。使用物种特异性的基因组和注释文件，您可以生成训练和测试数据集，然后训练一个专为牡蛎定制的剪接位点预测模型。该流程与人类 MANE 工作流程类似，但根据需要对输入文件和参数进行了调整。

前置条件

在开始之前，请确保您已经：

  1. 安装了 OpenSpliceAI：请按照安装页面的说明进行操作。
  2. 克隆了代码库：从 LiftOn OpenSpliceAI 代码库克隆 OpenSpliceAI 代码库。
  3. 下载了长牡蛎的输入文件：
    • 参考基因组：例如，长牡蛎基因组组装可以从这里获取 GCA_902806645.1_cgigas_uk_roslin_v1_genomic.fna
    • 注释文件：长牡蛎基因注释文件 GCF_902806645.1_cgigas_uk_roslin_v1_genomic.gff

步骤 1：检查输入文件

转换基因组名字，使其与注释gff文件中名字一致：

#!/bin/bash

# 变量配置
FLANKING_SIZE=400
EXPERIMENT_NAME="oyster_full_dataset"
TRAIN_DATASET="/home/hhhh/OpenSpliceAI/OpenSpliceAI/spliciai/train_test_dataset/dataset_train.h5"
TEST_DATASET="/home/hhhh/OpenSpliceAI/OpenSpliceAI/spliciai/train_test_dataset/dataset_test.h5"
OUTPUT_DIR="/home/hhhh/OpenSpliceAI/OpenSpliceAI/spliciai/oyster_model_train_outdir"
PROJECT_NAME="OpenSpliceAI-Oyster"
RANDOM_SEED=42
MODEL="SpliceAI"
LOSS="cross_entropy_loss"
EPOCHS=20
PATIENCE=5
SCHEDULER="MultiStepLR"

# 创建输出目录（如果不存在）
mkdir -p "$OUTPUT_DIR"

# 执行训练命令
openspliceai train \
   --flanking-size $FLANKING_SIZE \
   --exp-num $EXPERIMENT_NAME \
   --train-dataset $TRAIN_DATASET \
   --test-dataset $TEST_DATASET \
   --output-dir $OUTPUT_DIR \
   --project-name $PROJECT_NAME \
   --random-seed $RANDOM_SEED \
   --model $MODEL \
   --loss $LOSS \
   --epochs $EPOCHS \
   --patience $PATIENCE \
   --scheduler $SCHEDULER

echo "Training completed. Model and logs saved to $OUTPUT_DIR."

命名为1.n2n.sh，chmod +x 1.n2n.sh，./1.n2n.sh 运行命令。

步骤 2：创建训练和测试数据集

使用 create-data 子命令处理小鼠参考基因组和注释文件，将其转换为 HDF5 格式的数据集。此步骤提取基因序列，选择典型转录本（通常是蛋白质编码的），并对序列和剪接位点标签进行独热编码。

运行以下命令（请根据需要更新文件名）：

#!/bin/bash

GENOME="GCA_902806645.1_cgigas_uk_roslin_v1_genomic_fixed.fna"
ANNOTATION="GCF_902806645.1_cgigas_uk_roslin_v1_genomic.gff"
OUTPUT_DIR="train_test_dataset/"

mkdir -p ${OUTPUT_DIR}

echo "Starting accurate data processing at $(date)"
echo "This may take several hours for a complete genome..."

openspliceai create-data \
    --genome-fasta "$GENOME" \
    --annotation-gff "$ANNOTATION" \
    --output-dir "$OUTPUT_DIR" \
    --parse-type canonical \
    --split-method random \
    --remove-paralogs \
    --min-identity 0.8 \
    --min-coverage 0.8 \
    --canonical-only \
    --biotype protein-coding \
    --write-fasta \
    2>&1 | tee "$OUTPUT_DIR/processing.log"

echo "Completed at $(date)"
echo "Check $OUTPUT_DIR for output files"

关键选项解释：

  • --remove-paralogs：过滤掉旁系同源序列以避免数据泄露。
  • --min-identity 0.8 和 --min-coverage 0.8：设置过滤阈值。
  • --parse-type canonical：选择每个基因最长的典型转录本。
  • --split-method mouse：应用小鼠特定的规则将数据分割成训练集和测试集。
  • --canonical-only：将标记限制在保守的剪接位点模体。

运行此命令后，检查输出目录中的两个主要文件：

  • dataset_train.h5（训练数据集）
  • dataset_test.h5（测试数据集）

命名为2.create-data_cg.sh，chmod +x 2.create-data_cg.sh，./2.create-data_cg.sh 运行命令。

步骤 3：训练长牡蛎模型

数据集准备好后，使用 train 子命令训练您的小鼠剪接位点预测模型。您可以根据应用选择合适的两侧侧翼序列长度；在此示例中，我们使用 10,000 个核苷酸的侧翼大小，这在先前的研究中已被证明是有效的。

运行以下命令（根据需要调整文件路径和参数）：

#!/bin/bash

# 变量配置
FLANKING_SIZE=400
EXPERIMENT_NAME="oyster_full_dataset"
TRAIN_DATASET="/home/hhhh/OpenSpliceAI/OpenSpliceAI/spliciai/train_test_dataset/dataset_train.h5"
TEST_DATASET="/home/hhhh/OpenSpliceAI/OpenSpliceAI/spliciai/train_test_dataset/dataset_test.h5"
OUTPUT_DIR="/home/hhhh/OpenSpliceAI/OpenSpliceAI/spliciai/oyster_model_train_outdir"
PROJECT_NAME="OpenSpliceAI-Oyster"
RANDOM_SEED=42
MODEL="SpliceAI"
LOSS="cross_entropy_loss"
EPOCHS=20
PATIENCE=5
SCHEDULER="MultiStepLR"

# 创建输出目录（如果不存在）
mkdir -p "$OUTPUT_DIR"

# 执行训练命令
openspliceai train \
   --flanking-size $FLANKING_SIZE \
   --exp-num $EXPERIMENT_NAME \
   --train-dataset $TRAIN_DATASET \
   --test-dataset $TEST_DATASET \
   --output-dir $OUTPUT_DIR \
   --project-name $PROJECT_NAME \
   --random-seed $RANDOM_SEED \
   --model $MODEL \
   --loss $LOSS \
   --epochs $EPOCHS \
   --patience $PATIENCE \
   --scheduler $SCHEDULER

echo "Training completed. Model and logs saved to $OUTPUT_DIR."

选项解释：

  • --flanking-size 10000：将侧翼区域设置为 10,000 个核苷酸。
  • --exp-num mouse_full_dataset：此实验的标签。
  • --random-seed 42：确保结果可重现。
  • --model SpliceAI：使用 SpliceAI 模型架构。
  • --loss cross_entropy_loss：指定损失函数。
  • 模型检查点（例如 model_best.pt）和训练日志将保存在指定的输出目录中。

命名为3.train_model.sh，chmod +x 3.train_model.sh，./3.train_model.sh 运行命令。

步骤 4（可选）：转移模型

下面可以不运行，前面模型训练使用的fz == 400，机器不行，没跑大的，所以下面使用transfer训练2000的。

运行以下命令（根据需要调整文件路径和参数）：

openspliceai transfer \
   --train-dataset /home/hhhh/OpenSpliceAI/OpenSpliceAI/spliciai/train_test_dataset/dataset_train.h5 \
   --test-dataset /home/hhhh/OpenSpliceAI/OpenSpliceAI/spliciai/train_test_dataset/dataset_test.h5 \
   --pretrained-model /home/hhhh/OpenSpliceAI/OpenSpliceAI/spliciai/oyster_model_train_outdir/SpliceAI_OpenSpliceAI-Oyster_400_oyster_full_dataset_rs42/oyster_full_dataset/models/model_best.pt \
   --flanking-size 2000 \
   --unfreeze 2 \
   --scheduler CosineAnnealingWarmRestarts \
   --loss focal_loss \
   --epochs 10 \
   --patience 3 \
   --early-stopping \
   --project-name OpenSpliceAI-Oyster_2000_transfer \
   --output-dir /home/hhhh/OpenSpliceAI/OpenSpliceAI/spliciai/oyster_model_train_outdir

命名为4.transfer_model.sh，chmod +x 4.transfer_model.sh，./4.transfer_model.sh 运行命令。

步骤 5（可选）：校准长牡蛎模型

校准可以优化模型的输出概率，使其更能反映经验上的剪接位点可能性。如果需要，可以使用 calibrate 子命令运行校准步骤。

执行：

# 强制使用CPU避免内存问题
export CUDA_VISIBLE_DEVICES=""

openspliceai calibrate \
   --flanking-size 2000 \
   --train-dataset train_test_dataset/dataset_train.h5 \
   --test-dataset train_test_dataset/dataset_test.h5 \
   --output-dir model_calibrate_outdir_oyster/ \
   --project-name OpenSpliceAI-oyster-calibrate \
   --random-seed 42 \
   --pretrained-model oyster_model_train_outdir/SpliceAI_OpenSpliceAI-Oyster_2000_transfer_2000_0_rs42/0/models/model_best.pt \
   --loss cross_entropy_loss

此命令：

  • 从您的长牡蛎训练运行中加载最佳模型检查点。
  • 优化用于校准的温度参数。
  • 将校准输出（例如，温度参数、可靠性曲线）保存在指定目录中。

步骤 6：部署您训练好的长牡蛎模型

训练（以及可选的校准）完成后，您的小鼠模型就可以使用了。您现在可以：

  • 预测剪接位点：使用 predict 子命令对新的长牡蛎 FASTA 序列进行推理。
  • 分析变异效应：使用 variant 子命令评估特定突变如何影响长牡蛎基因组中的剪接。

import re
import sys

def extract_chromosome_lengths(fasta_file, output_file):
    print(f"从FASTA文件提取染色体长度: {fasta_file}")
    
    chrom_lengths = {}
    current_chrom = None
    current_length = 0
    
    with open(fasta_file, 'r') as f:
        for line in f:
            if line.startswith('>'):
                # 保存前一个染色体的长度
                if current_chrom and current_length > 0:
                    chrom_lengths[current_chrom] = current_length
                
                # 解析新的染色体头
                header = line[1:].strip()
                # 提取染色体名称（通常是第一个空格前的部分）
                chrom_name = header.split()[0]
                current_chrom = chrom_name
                current_length = 0
                
                print(f"找到染色体: {chrom_name}")
            else:
                # 计算序列长度
                current_length += len(line.strip())
        
        # 保存最后一个染色体
        if current_chrom and current_length > 0:
            chrom_lengths[current_chrom] = current_length
    
    print(f"\n📊 染色体长度统计:")
    for chrom, length in chrom_lengths.items():
        print(f"  {chrom}: {length:,} bp")
    
    # 写入VCF头文件
    with open(output_file, 'w') as out:
        out.write("##fileformat=VCFv4.2\n")
        out.write('##INFO=<ID=SpliceAI,Number=.,Type=String,Description="OpenSpliceAI annotation">\n')
        
        # 写入准确的contig定义
        for chrom, length in chrom_lengths.items():
            out.write(f'##contig=<ID={chrom},length={length}>\n')
        
        out.write("#CHROM\tPOS\tID\tREF\tALT\tQUAL\tFILTER\tINFO\n")
    
    print(f"✅ VCF头文件已生成: {output_file}")
    return chrom_lengths

if __name__ == "__main__":
    if len(sys.argv) != 3:
        print("用法: python extract_chrom_lengths.py input.fasta output.vcf_header")
        sys.exit(1)
    
    chrom_lengths = extract_chromosome_lengths(sys.argv[1], sys.argv[2])
EOF

# 运行染色体长度提取

cat > "${OUTPUT_DIR}/fix_gff_with_filter.py" << 'EOF'
import pandas as pd
import re
import sys

def extract_gene_info_with_filter(gff_file, output_file):
    print("读取GFF文件并提取完整基因信息（过滤无外显子基因）...")
    
    genes = {}
    mrnas = {}
    exons = []
    
    with open(gff_file, 'r') as f:
        for line in f:
            if line.startswith('#'):
                continue
            
            fields = line.strip().split('\t')
            if len(fields) < 9:
                continue
            
            chrom, source, feature_type, start, end, score, strand, phase, attributes = fields
            
            if feature_type == 'gene':
                gene_id = 'Unknown'
                gene_name = 'Unknown'
                
                if 'ID=' in attributes:
                    gene_id = attributes.split('ID=')[1].split(';')[0]
                if 'gene=' in attributes:
                    gene_name = attributes.split('gene=')[1].split(';')[0]
                elif 'Name=' in attributes:
                    gene_name = attributes.split('Name=')[1].split(';')[0]
                elif gene_id != 'Unknown':
                    gene_name = gene_id
                
                genes[gene_id] = {
                    'name': gene_name,
                    'chrom': chrom,
                    'strand': strand,
                    'start': int(start),
                    'end': int(end),
                    'mrnas': []
                }
            
            elif feature_type == 'mRNA':
                mrna_id = 'Unknown'
                parent_gene = 'Unknown'
                
                if 'ID=' in attributes:
                    mrna_id = attributes.split('ID=')[1].split(';')[0]
                if 'Parent=' in attributes:
                    parent_gene = attributes.split('Parent=')[1].split(';')[0]
                
                mrnas[mrna_id] = {
                    'parent': parent_gene,
                    'chrom': chrom,
                    'strand': strand,
                    'start': int(start),
                    'end': int(end),
                    'exons': []
                }
                
                if parent_gene in genes:
                    genes[parent_gene]['mrnas'].append(mrna_id)
            
            elif feature_type == 'exon':
                parent_mrna = 'Unknown'
                
                if 'Parent=' in attributes:
                    parent_mrna = attributes.split('Parent=')[1].split(';')[0]
                
                exons.append({
                    'parent': parent_mrna,
                    'chrom': chrom,
                    'start': int(start),
                    'end': int(end)
                })
    
    print(f"找到 {len(genes)} 个基因, {len(mrnas)} 个mRNA, {len(exons)} 个外显子")
    
    # 关联外显子到mRNA
    for exon in exons:
        parent = exon['parent']
        if parent in mrnas:
            mrnas[parent]['exons'].append((exon['start'], exon['end']))
    
    # 为每个基因选择最长的mRNA作为代表，并过滤无外显子的基因
    print("处理基因和转录本（过滤无外显子基因）...")
    gene_records = []
    skipped_genes = 0
    
    for gene_id, gene_info in genes.items():
        if not gene_info['mrnas']:
            skipped_genes += 1
            continue
        else:
            longest_mrna = None
            max_length = 0
            
            for mrna_id in gene_info['mrnas']:
                if mrna_id in mrnas:
                    mrna = mrnas[mrna_id]
                    length = mrna['end'] - mrna['start']
                    if length > max_length:
                        max_length = length
                        longest_mrna = mrna
            
            if longest_mrna and longest_mrna['exons']:
                sorted_exons = sorted(longest_mrna['exons'], key=lambda x: x[0])
                exon_starts = [exon[0] for exon in sorted_exons]
                exon_ends = [exon[1] for exon in sorted_exons]
                
                gene_records.append({
                    'name': gene_info['name'],
                    'chrom': gene_info['chrom'],
                    'strand': gene_info['strand'],
                    'tx_start': longest_mrna['start'],
                    'tx_end': longest_mrna['end'],
                    'exon_starts': exon_starts,
                    'exon_ends': exon_ends
                })
            else:
                skipped_genes += 1
    
    print(f"生成 {len(gene_records)} 个基因记录")
    print(f"跳过 {skipped_genes} 个无外显子信息的基因")
    
    # 写入输出文件
    print("写入输出文件...")
    with open(output_file, 'w') as out:
        out.write("#NAME\tCHROM\tSTRAND\tTX_START\tTX_END\tEXON_START\tEXON_END\n")
        
        for gene in gene_records:
            exon_start_str = ','.join(map(str, gene['exon_starts'])) + ','
            exon_end_str = ','.join(map(str, gene['exon_ends'])) + ','
            
            out.write(f"{gene['name']}\t{gene['chrom']}\t{gene['strand']}\t")
            out.write(f"{gene['tx_start']}\t{gene['tx_end']}\t")
            out.write(f"{exon_start_str}\t{exon_end_str}\n")
    
    print(f"转换完成: {output_file}")
    
    print(f"\n📊 统计信息:")
    print(f"总基因数: {len(genes)}")
    print(f"有效基因数（有外显子）: {len(gene_records)}")
    print(f"无效基因数（无外显子）: {skipped_genes}")
    print(f"过滤率: {skipped_genes/len(genes)*100:.1f}%")

if __name__ == "__main__":
    if len(sys.argv) != 3:
        print("用法: python fix_gff_with_filter.py input.gff output.txt")
        sys.exit(1)
    
    extract_gene_info_with_filter(sys.argv[1], sys.argv[2])
EOF

python3 "${OUTPUT_DIR}/fix_gff_with_filter.py" "${INPUT_GFF}" "${OUTPUT_TXT}"

import sys
import re

vcf_file = sys.argv[1]
output_file = sys.argv[2]

with open(vcf_file, 'r') as f, open(output_file, 'w') as out:
    out.write("Chromosome\tPosition\tRef\tAlt\tGene\tDS_AG\tDS_AL\tDS_DG\tDS_DL\tMax_Score\tImpact\n")
    
    count = 0
    for line in f:
        if line.startswith('#'):
            continue
        
        fields = line.strip().split('\t')
        if len(fields) < 8:
            continue
        
        chrom, pos, _, ref, alt, _, _, info = fields
        
        if 'SpliceAI=' in info:
            match = re.search(r'SpliceAI=([^;]+)', info)
            if match:
                data = match.group(1).split('|')
                if len(data) >= 7:
                    gene = data[1]
                    try:
                        ds_ag = float(data[2]) if data[2] != '.' else 0.0
                        ds_al = float(data[3]) if data[3] != '.' else 0.0
                        ds_dg = float(data[4]) if data[4] != '.' else 0.0
                        ds_dl = float(data[5]) if data[5] != '.' else 0.0
                        
                        max_score = max(ds_ag, ds_al, ds_dg, ds_dl)
                        
                        if max_score > 0.8:
                            impact = "VERY_HIGH"
                        elif max_score > 0.5:
                            impact = "HIGH"
                        elif max_score > 0.2:
                            impact = "MEDIUM"
                        else:
                            impact = "LOW"
                        
                        out.write(f"{chrom}\t{pos}\t{ref}\t{alt}\t{gene}\t{ds_ag:.6f}\t{ds_al:.6f}\t{ds_dg:.6f}\t{ds_dl:.6f}\t{max_score:.6f}\t{impact}\n")
                        count += 1
                    except ValueError:
                        continue

print(f"提取了 {count} 个变异的结果")
EOF

python3 "${OUTPUT_DIR}/extract_results.py" "${OUTPUT_VCF}" "${FINAL_RESULTS}"

# 6. 清理和报告
echo "[6/6] 清理和生成报告..."
rm -f "${OUTPUT_DIR}/extract_chrom_lengths.py" "${OUTPUT_DIR}/fix_gff_with_filter.py" "${OUTPUT_DIR}/extract_results.py" "${OUTPUT_DIR}/vcf_header.txt"

# 显示染色体长度信息
echo ""
echo "📏 使用的准确染色体长度:"
python3 << EOF
import re

with open("${REF_GENOME}", 'r') as f:
    chrom_lengths = {}
    current_chrom = None
    current_length = 0
    
    for line in f:
        if line.startswith('>'):
            if current_chrom and current_length > 0:
                chrom_lengths[current_chrom] = current_length
            
            header = line[1:].strip()
            chrom_name = header.split()[0]
            current_chrom = chrom_name
            current_length = 0
        else:
            current_length += len(line.strip())
    
    if current_chrom and current_length > 0:
        chrom_lengths[current_chrom] = current_length

print("染色体长度统计:")
for chrom, length in sorted(chrom_lengths.items()):
    print(f"  {chrom}: {length:,} bp")

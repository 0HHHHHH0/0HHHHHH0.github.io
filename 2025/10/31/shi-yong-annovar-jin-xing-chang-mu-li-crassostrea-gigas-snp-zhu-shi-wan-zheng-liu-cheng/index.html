使用ANNOVAR进行长牡蛎(Crassostrea gigas)SNP注释完整流程

概述

本文介绍如何使用ANNOVAR对长牡蛎(Crassostrea gigas)的SNP变异数据进行功能注释。流程包括参考基因组数据库构建、变异数据格式转换、功能注释和结果整合。

环境准备

1. 软件安装

# ANNOVAR下载
wget http://www.openbioinformatics.org/annovar/annovar.tar.gz
tar -xzvf annovar.tar.gz

# 其他依赖工具
conda install -c bioconda ucsc-gtftogenepred

2. 数据文件准备

  • 参考基因组GTF文件：/home/wanglab/Ywh/adductor_muscle_scar/Crassostrea_gigas.cgigas_uk_roslin_v1.61.gtf
  • 基因组FASTA文件：GCF_902806645.1_cgigas_uk_roslin_v1_genomic.fna
  • SNP数据：filtered_variants_thr0p7_20250902_162128.tsv

完整流程

步骤1：构建自定义参考基因组数据库

1.1 转换GTF为GenePred格式

gtfToGenePred -genePredExt \
    "/home/wanglab/Ywh/adductor_muscle_scar/ReSeq_Data/DZQD2023092909/annovar/Cgigasdb/GCF_902806645.1_cgigas_uk_roslin_v1_genomic.gtf" \
    Cg_refGene.txt

1.2 提取mRNA序列

perl retrieve_seq_from_fasta.pl \
    --format refGene \
    --seqfile GCF_902806645.1_cgigas_uk_roslin_v1_genomic.fna \
    --outfile Cg_refGeneMrna.fa \
    Cg_refGene.txt

步骤2：准备SNP输入文件

2.1 转换TSV为ANNOVAR输入格式

awk 'NR>1 {print $1"\t"$2"\t"$2"\t"$3"\t"$4}' \
    /home/wanglab/Ywh/adductor_muscle_scar/Variants/filtered_variants_thr0p7_20250902_162128.tsv \
    > filtered_variants_thr0p7_20250902_162128.avinput

输入TSV格式：

chromosome	position	ref	alt	...
NC_047567.1	32667511	G	T	...

输出AVINPUT格式：

chromosome	start	end	ref	alt
NC_047567.1	32667511	32667511	G	T

步骤3：执行功能注释

3.1 使用table_annovar.pl进行注释

perl table_annovar.pl \
    filtered_variants_thr0p7_20250902_162128.avinput \
    Cgigasdb/ \
    -buildver Cg \
    -out filtered_variants_thr0p7_20250902_162128 \
    -remove \
    -protocol refGene \
    -operation g \
    -nastring . \
    -csvout

参数说明：

  • -buildver Cg: 使用自定义构建版本”Cg”
  • -protocol refGene: 使用refGene数据库
  • -operation g: 基因-based注释
  • -csvout: 输出CSV格式
  • -nastring .: 用”.”表示缺失值

步骤4：整合注释结果

4.1 方法一：基于行号合并（推荐）

# 为原始文件添加行号
awk 'NR==1 {print "LineNumber\t"$0} NR>1 {print NR-1"\t"$0}' \
    /home/wanglab/Ywh/adductor_muscle_scar/Variants/filtered_variants_thr0p7_20250902_162128.tsv \
    > original_with_linenum.tsv

# 为注释文件添加行号
awk 'NR==1 {print "LineNumber\t"$0} NR>1 {print NR-1"\t"$0}' \
    filtered_variants_thr0p7_20250902_162128.Cg_multianno.csv \
    > anno_with_linenum.tsv

# 基于行号合并
join -t $'\t' -1 1 -2 1 \
    original_with_linenum.tsv anno_with_linenum.tsv \
    > merged_result.tsv

# 清理临时文件
rm original_with_linenum.tsv anno_with_linenum.tsv

4.2 方法二：一次性合并

awk -F',' '
BEGIN {OFS="\t"}
NR==FNR {
    if (FNR==1) {
        anno_header = $6"\t"$7"\t"$8"\t"$9"\t"$10
    } else {
        annotation[FNR] = $6"\t"$7"\t"$8"\t"$9"\t"$10
    }
    next
}
{
    if (FNR==1) {
        print $0 "\t" anno_header
    } else {
        print $0 "\t" annotation[FNR]
    }
}' filtered_variants_thr0p7_20250902_162128.Cg_multianno.csv \
   /home/wanglab/Ywh/adductor_muscle_scar/Variants/filtered_variants_thr0p7_20250902_162128.tsv \
   > filtered_variants_thr0p7_20250902_162128_annotated.tsv

输出结果解读

注释结果包含的字段：

  • Gene-based注释: 基因名称、转录本、外显子功能等
  • 位置信息: 染色体、位置、参考/变异碱基
  • 功能预测: 对蛋白质功能的影响

典型输出格式：

Chr     Start   End     Ref     Alt     Func.refGene    Gene.refGene    ExonicFunc.refGene    AAChange.refGene
NC_047567.1 32667511 32667511 G T downstream LOC105320120 . .
NC_047567.1 32661831 32661831 C A exonic LOC105320120 nonsynonymous SNV LOC105320120:NM_001345123:exon2:c.T142A:p.S48T

常见问题与解决方案

1. 数据库构建失败

问题: GTF文件格式不兼容
解决: 确保GTF文件来自官方来源，使用gtfToGenePred前检查文件完整性

2. 注释结果为空

问题: 构建版本不匹配
解决: 检查-buildver参数与数据库构建版本一致

3. 合并结果错位

问题: 输入文件行数不匹配
解决: 使用基于行号的合并方法确保准确性

流程优化建议

1. 批量处理

对于多个样本，可以编写循环脚本批量处理：

for sample in sample1 sample2 sample3; do
    perl table_annovar.pl ${sample}.avinput Cgigasdb/ \
        -buildver Cg \
        -out ${sample}_annotated \
        -protocol refGene \
        -operation g \
        -csvout
done

2. 添加更多注释数据库

# 添加保守性预测
perl table_annovar.pl input.avinput Cgigasdb/ \
    -buildver Cg \
    -out output \
    -protocol refGene,phastConsElements100way \
    -operation g,r \
    -csvout

本流程提供了从原始SNP数据到功能注释的完整解决方案，特别针对长牡蛎等非模式生物。通过构建自定义数据库，可以有效地对物种特异性变异进行功能注释，为后续的群体遗传学和功能基因组学研究奠定基础。

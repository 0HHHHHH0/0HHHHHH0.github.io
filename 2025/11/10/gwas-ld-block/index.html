GWAS + 热点区域识别 + LD Block 分析流程详解

🧬 一、项目概述

在进行 GWAS（全基因组关联分析） 后，我们通常得到一个包含数十万到数百万个 SNP 的结果文件。然而，仅凭曼哈顿图观察显著位点往往无法直观判断“显著 SNP 是否存在聚集（hotspot）”，更难直接选定 LD 分析区域。

本篇文章介绍一套自动化分析流程，用于：

  1. 从 GWAS 注释文件中提取特定染色体；
  2. 对 SNP 分布进行可视化与统计；
  3. 基于密度与显著性筛选“热点区域”；
  4. 输出推荐的 LD block 分析区域；
  5. 生成一整套图形报告（含基础可视化、热点分布、显著SNP分布、综合分析图）。

该流程适用于任何 GWAS 结果（如 .assoc 或 .anno.xls 文件），并以染色体 NC_047567.1 为示例展示。

⚙️ 二、环境配置

分析依赖 R 及多个可视化与基因组处理包，可通过以下命令一次性安装：

# 一键安装所有依赖

✅ 安装完成后即可运行所有 R 脚本，无需额外配置。

2️⃣ 提取单条染色体数据

2.extract_NC_047567.1.sh

#!/bin/bash
CHROM="NC_047567.1"
ANNO_FILE="color_GWAS.anno.xls"

echo "=== 提取染色体 $CHROM 数据 ==="

# 提取表头
head -n 1 "$ANNO_FILE" > "${CHROM}_data.txt"

# 提取该染色体的数据
awk -v chrom="$CHROM" '$1 == chrom' "$ANNO_FILE" >> "${CHROM}_data.txt"

# 统计
TOTAL_LINES=$(wc -l < "${CHROM}_data.txt")
SNP_COUNT=$((TOTAL_LINES - 1))

echo "提取完成:"
echo "- 输出文件: ${CHROM}_data.txt"
echo "- SNP数量: $SNP_COUNT"

# 检查文件内容
echo ""
echo "前3行数据:"
head -n 3 "${CHROM}_data.txt"

# 生成位置文件
awk 'NR>1 {print $2}' "${CHROM}_data.txt" > "${CHROM}_positions.txt"
echo "位置文件: ${CHROM}_positions.txt"

# 位置范围统计
echo ""
echo "位置范围统计:"
awk 'NR>1 {print $2}' "${CHROM}_data.txt" | sort -n | awk '
BEGIN {min=999999999; max=0} 
{count++; sum+=$1; if($1<min)min=$1; if($1>max)max=$1} 
END {
    print "最小位置: " min
    print "最大位置: " max  
    print "位置范围: " min " - " max
    print "总跨度: " (max-min)/1000000 " Mb"
    print "SNP数量: " count
}'

extract_NC_047567.1.sh 用于从注释表中提取指定染色体 SNP 信息。

核心步骤：

  • 从原始 GWAS 注释文件中筛选出对应染色体；
  • 输出该染色体的 SNP 表；
  • 自动生成位置文件与区间统计。

输出：

- 输出文件: NC_047567.1_data.txt
- SNP数量: 2345
- 位置范围: 1.2 Mb - 39.7 Mb
- 总跨度: 38.5 Mb

📈 四、R 脚本分析模块

🔹 1. 完整基础分析（3.0.complete_analysis.R）

3.0.complete_analysis.R

#!/usr/bin/env Rscript
cat("=== 完整的染色体SNP分析 ===\n")

library(ggplot2)
library(dplyr)
library(viridis)
library(patchwork)

# 读取数据
chrom <- "NC_047567.1"
snp_data <- read.table(paste0(chrom, "_data.txt"), header = TRUE, sep = "\t", stringsAsFactors = FALSE)
snp_data$logP <- -log10(snp_data$P.value)

cat("数据概况:\n")
cat("- SNP总数:", format(nrow(snp_data), big.mark = ","), "\n")
cat("- 位置范围:", format(min(snp_data$POS), big.mark = ","), "-", format(max(snp_data$POS), big.mark = ","), "\n")
cat("- 染色体长度:", format((max(snp_data$POS) - min(snp_data$POS))/1000000, digits=2), "Mb\n")
cat("- P值范围:", min(snp_data$P.value), "-", max(snp_data$P.value), "\n")

# 1. 基础可视化
cat("生成基础可视化...\n")

# 密度图
p_density <- ggplot(snp_data, aes(x = POS/1000000)) +
    geom_density(fill = "steelblue", alpha = 0.7) +
    labs(title = "SNP密度分布", x = "基因组位置 (Mb)", y = "密度") +
    theme_minimal()

# 曼哈顿图
p_manhattan <- ggplot(snp_data, aes(x = POS/1000000, y = logP)) +
    geom_point(aes(color = logP), alpha = 0.3, size = 0.5) +
    scale_color_viridis_c(name = "-log10(P)") +
    geom_hline(yintercept = -log10(0.05/nrow(snp_data)), linetype = "dashed", color = "red") +
    labs(title = "曼哈顿图", x = "基因组位置 (Mb)", y = "-log10(P值)") +
    theme_minimal()

# 组合图
png(paste0(chrom, "_basic_analysis.png"), width=16, height=10, units="in", res=300)
basic_plot <- p_density / p_manhattan +
    plot_annotation(
        title = paste("染色体", chrom, "SNP分析"),
        subtitle = paste("总计", format(nrow(snp_data), big.mark = ","), "个SNP"),
        theme = theme(plot.title = element_text(hjust = 0.5, size=16, face="bold"))
    )
print(basic_plot)
dev.off()

# 2. 正确的热点区域分析
cat("进行热点区域分析...\n")

# 使用固定大小的窗口（1Mb窗口）
window_size <- 1000000  # 1Mb窗口
starts <- seq(min(snp_data$POS), max(snp_data$POS), by = window_size)

hotspot_analysis <- data.frame()
for(i in 1:(length(starts)-1)) {
    start <- starts[i]
    end <- starts[i+1]
    window_data <- snp_data[snp_data$POS >= start & snp_data$POS < end, ]
    
    if(nrow(window_data) > 0) {
        hotspot_analysis <- rbind(hotspot_analysis, data.frame(
            Window = i,
            Start = start,
            End = end,
            SNP_Count = nrow(window_data),
            SNP_Density = nrow(window_data) / ((end - start)/1000000),  # 每Mb的SNP数
            Mean_logP = mean(window_data$logP, na.rm = TRUE),
            Max_logP = max(window_data$logP, na.rm = TRUE)
        ))
    }
}

cat("窗口分析完成，窗口数量:", nrow(hotspot_analysis), "\n")
cat("平均每窗口SNP数量:", round(mean(hotspot_analysis$SNP_Count)), "\n")
cat("平均SNP密度:", round(mean(hotspot_analysis$SNP_Density)), "SNPs/Mb\n")

# 识别热点区域（SNP密度前20%）
density_threshold <- quantile(hotspot_analysis$SNP_Density, 0.80)
hot_regions <- hotspot_analysis %>%
    filter(SNP_Density >= density_threshold) %>%
    arrange(desc(SNP_Density))

cat("热点区域阈值:", round(density_threshold), "SNPs/Mb\n")
cat("发现热点区域:", nrow(hot_regions), "个\n")

# 热点区域可视化
png(paste0(chrom, "_hotspot_corrected.png"), width=14, height=6, units="in", res=300)
ggplot(hotspot_analysis, aes(x = Start/1000000, y = SNP_Density)) +
    geom_col(aes(fill = SNP_Density), alpha = 0.8, width = 0.8) +
    geom_hline(yintercept = density_threshold, linetype = "dashed", color = "red", size = 1) +
    geom_text(data = hot_regions, aes(label = round(SNP_Density)), vjust = -0.5, size = 3) +
    scale_fill_viridis_c(name = "SNP密度\n(SNPs/Mb)") +
    labs(title = paste("SNP热点区域分析 -", chrom),
         subtitle = paste("红色虚线: 热点阈值 (", round(density_threshold), "SNPs/Mb)"),
         x = "基因组位置 (Mb)", y = "SNP密度 (SNPs/Mb)") +
    theme_minimal() +
    theme(plot.title = element_text(hjust = 0.5, face = "bold"))
dev.off()

# 3. 输出热点区域和推荐分析区域
if(nrow(hot_regions) > 0) {
    hotspot_info <- data.frame(
        Region_ID = paste0("Hotspot_", 1:nrow(hot_regions)),
        Chromosome = chrom,
        Start_Mb = round(hot_regions$Start/1000000, 1),
        End_Mb = round(hot_regions$End/1000000, 1),
        Size_Mb = 1.0,  # 固定1Mb窗口
        SNP_Count = hot_regions$SNP_Count,
        SNP_Density = round(hot_regions$SNP_Density),
        Mean_logP = round(hot_regions$Mean_logP, 2),
        Max_logP = round(hot_regions$Max_logP, 2)
    )
    
    write.csv(hotspot_info, paste0(chrom, "_corrected_hotspots.csv"), row.names = FALSE)
    
    # 推荐前3个区域进行LD分析
    recommended <- head(hotspot_info, 3)
    write.csv(recommended, paste0(chrom, "_recommended_ld_regions.csv"), row.names = FALSE)
    
    cat("\n=== 热点区域总结 ===\n")
    print(hotspot_info)
    
    cat("\n=== 推荐LD分析区域 ===\n")
    print(recommended)
} else {
    cat("未发现明显热点区域\n")
}

# 4. 生成分析报告
cat("\n=== 分析完成 ===\n")
cat("生成的文件:\n")
cat("- ", chrom, "_basic_analysis.png (基础分析图)\n")
cat("- ", chrom, "_hotspot_corrected.png (热点区域图)\n")
cat("- ", chrom, "_corrected_hotspots.csv (热点区域列表)\n")
cat("- ", chrom, "_recommended_ld_regions.csv (推荐LD分析区域)\n")

cat("\n下一步建议:\n")
if(nrow(hot_regions) > 0) {
    cat("1. 对推荐区域进行LD block分析\n")
    cat("2. 重点关注高SNP密度且P值显著的区域\n")
    cat("3. 结合基因注释分析候选基因\n")
} else {
    cat("1. 考虑分析整个染色体或选择SNP密度较高的区域\n")
    cat("2. 可以手动选择感兴趣的区域进行LD分析\n")
}

该脚本是整个流程的核心，包含以下分析步骤：

（1）基础可视化

生成 SNP密度图 与 曼哈顿图：

  • 横轴为基因组位置（Mb）；
  • 纵轴为 -log10(P)；
  • 自动添加 Bonferroni 阈值线；
  • 图像合并保存为 *_basic_analysis.png。

（2）窗口扫描与热点检测

通过 1Mb 窗口滑动统计每 Mb 的 SNP 数量与密度，并识别：

  • SNP 密度位于前 20% 的窗口；
  • 输出为 *_corrected_hotspots.csv；
  • 可视化结果为 *_hotspot_corrected.png。

（3）推荐 LD 分析区域

程序自动输出密度最高的前三个窗口，保存为：

  • *_recommended_ld_regions.csv

（4）结果总结与建议

输出包括：

=== 热点区域总结 ===
=== 推荐LD分析区域 ===
下一步建议:
1. 对推荐区域进行LD block分析
2. 结合基因注释分析候选基因

🔹 2. 基于显著SNP的热点分析（3.1.corrected_significant_hotspots.R）

3.1.corrected_significant_hotspots.R

#!/usr/bin/env Rscript
cat("=== 基于显著SNP的热点区域分析 ===\n")

library(ggplot2)
library(dplyr)
library(viridis)
library(patchwork)

# 读取数据
chrom <- "NC_047567.1"
snp_data <- read.table(paste0(chrom, "_data.txt"), header = TRUE, sep = "\t", stringsAsFactors = FALSE)
snp_data$logP <- -log10(snp_data$P.value)

cat("原始数据概况:\n")
cat("- 总SNP数:", format(nrow(snp_data), big.mark = ","), "\n")

# 定义显著性阈值
significance_threshold <- -log10(0.05/nrow(snp_data))  # Bonferroni校正
cat("Bonferroni校正的显著性阈值: -log10(P) >", round(significance_threshold, 2), "\n")

# 过滤显著SNP
significant_snps <- snp_data[snp_data$logP > significance_threshold, ]
cat("显著SNP数量:", format(nrow(significant_snps), big.mark = ","), "\n")
cat("显著SNP比例:", round(nrow(significant_snps)/nrow(snp_data)*100, 2), "%\n")

if(nrow(significant_snps) == 0) {
    cat("警告: 没有显著SNP，使用较宽松的阈值\n")
    significance_threshold <- -log10(1e-5)  # 使用10^-5作为阈值
    significant_snps <- snp_data[snp_data$logP > significance_threshold, ]
    cat("使用阈值 -log10(P) >", significance_threshold, "\n")
    cat("显著SNP数量:", nrow(significant_snps), "\n")
}

if(nrow(significant_snps) > 0) {
    # 1. 显著SNP的分布可视化
    p1 <- ggplot(significant_snps, aes(x = POS/1000000, y = logP)) +
        geom_point(aes(color = logP), alpha = 0.7, size = 1) +
        scale_color_viridis_c(name = "-log10(P)") +
        geom_hline(yintercept = significance_threshold, linetype = "dashed", color = "red") +
        labs(title = "显著SNP分布",
             x = "基因组位置 (Mb)", y = "-log10(P值)") +
        theme_minimal()
    
    p2 <- ggplot(significant_snps, aes(x = POS/1000000)) +
        geom_density(fill = "red", alpha = 0.7) +
        labs(title = "显著SNP密度分布",
             x = "基因组位置 (Mb)", y = "密度") +
        theme_minimal()
    
    png(paste0(chrom, "_significant_snps.png"), width=16, height=8, units="in", res=300)
    combined <- p1 | p2 +
        plot_annotation(
            title = paste("显著SNP分析 -", chrom),
            subtitle = paste("显著SNP:", nrow(significant_snps), "/", format(nrow(snp_data), big.mark = ",")),
            theme = theme(plot.title = element_text(hjust = 0.5, face="bold"))
        )
    print(combined)
    dev.off()
    
    # 2. 基于显著SNP的热点区域分析
    cat("进行显著SNP热点区域分析...\n")
    
    # 使用500kb窗口分析显著SNP
    window_size <- 500000  # 500kb窗口
    starts <- seq(min(snp_data$POS), max(snp_data$POS), by = window_size)
    
    hotspot_analysis <- data.frame()
    for(i in 1:(length(starts)-1)) {
        start <- starts[i]
        end <- starts[i+1]
        window_data <- significant_snps[significant_snps$POS >= start & significant_snps$POS < end, ]
        
        if(nrow(window_data) > 0) {
            hotspot_analysis <- rbind(hotspot_analysis, data.frame(
                Window = i,
                Start = start,
                End = end,
                Significant_SNP_Count = nrow(window_data),
                SNP_Density = nrow(window_data) / ((end - start)/1000000),  # 每Mb的显著SNP数
                Mean_logP = mean(window_data$logP, na.rm = TRUE),
                Max_logP = max(window_data$logP, na.rm = TRUE)
            ))
        } else {
            # 也记录没有显著SNP的窗口
            hotspot_analysis <- rbind(hotspot_analysis, data.frame(
                Window = i,
                Start = start,
                End = end,
                Significant_SNP_Count = 0,
                SNP_Density = 0,
                Mean_logP = NA,
                Max_logP = NA
            ))
        }
    }
    
    cat("窗口分析完成，窗口数量:", nrow(hotspot_analysis), "\n")
    
    # 识别显著SNP热点区域（显著SNP密度前20%）
    if(sum(hotspot_analysis$Significant_SNP_Count > 0) > 0) {
        density_threshold <- quantile(hotspot_analysis$SNP_Density[hotspot_analysis$SNP_Density > 0], 0.80, na.rm = TRUE)
        hot_regions <- hotspot_analysis %>%
            filter(SNP_Density >= density_threshold) %>%
            arrange(desc(SNP_Density))
        
        cat("显著SNP热点阈值:", round(density_threshold, 1), "显著SNPs/Mb\n")
        cat("发现显著SNP热点区域:", nrow(hot_regions), "个\n")
        
        # 热点区域可视化
        png(paste0(chrom, "_significant_hotspots.png"), width=14, height=6, units="in", res=300)
        ggplot(hotspot_analysis, aes(x = Start/1000000, y = SNP_Density)) +
            geom_col(aes(fill = SNP_Density), alpha = 0.8, width = 0.4) +
            geom_hline(yintercept = density_threshold, linetype = "dashed", color = "red", linewidth = 1) +
            geom_text(data = hot_regions, aes(label = round(SNP_Density, 1)), vjust = -0.5, size = 3) +
            scale_fill_viridis_c(name = "显著SNP密度\n(SNPs/Mb)") +
            labs(title = paste("显著SNP热点区域分析 -", chrom),
                 subtitle = paste("红色虚线: 热点阈值 (", round(density_threshold, 1), "显著SNPs/Mb)"),
                 x = "基因组位置 (Mb)", y = "显著SNP密度 (SNPs/Mb)") +
            theme_minimal() +
            theme(plot.title = element_text(hjust = 0.5, face = "bold"))
        dev.off()
        
        # 输出热点区域
        if(nrow(hot_regions) > 0) {
            hotspot_info <- data.frame(
                Region_ID = paste0("SigHotspot_", 1:nrow(hot_regions)),
                Chromosome = chrom,
                Start_Mb = round(hot_regions$Start/1000000, 1),
                End_Mb = round(hot_regions$End/1000000, 1),
                Size_Mb = 0.5,  # 500kb窗口
                Significant_SNP_Count = hot_regions$Significant_SNP_Count,
                SNP_Density = round(hot_regions$SNP_Density, 1),
                Mean_logP = round(hot_regions$Mean_logP, 2),
                Max_logP = round(hot_regions$Max_logP, 2)
            )
            
            write.csv(hotspot_info, paste0(chrom, "_significant_hotspots.csv"), row.names = FALSE)
            
            # 推荐前3个区域进行LD分析
            recommended <- head(hotspot_info, 3)
            write.csv(recommended, paste0(chrom, "_recommended_significant_regions.csv"), row.names = FALSE)
            
            cat("\n=== 显著SNP热点区域 ===\n")
            print(hotspot_info)
            
            cat("\n=== 推荐LD分析区域（基于显著SNP）===\n")
            print(recommended)
        }
    } else {
        cat("警告: 没有发现显著SNP聚集的区域\n")
    }
    
} else {
    cat("错误: 没有显著SNP，无法进行热点区域分析\n")
}

cat("\n=== 分析完成 ===\n")

此脚本进一步聚焦于显著 SNP：

（1）显著性筛选

  • 默认使用 Bonferroni 阈值；
  • 若无显著 SNP，自动使用更宽松阈值（如 1e-5）；
  • 输出显著 SNP 的数量、比例及分布。

（2）基于显著SNP的窗口统计

采用更精细的 500kb 窗口滑动，计算：

  • 显著SNP数；
  • 显著SNP密度；
  • 平均与最大 -log10(P)。

筛选密度前 20% 的窗口作为 显著热点区域，输出为：

  • *_significant_hotspots.csv
  • 推荐区域：*_recommended_significant_regions.csv

（3）显著SNP热点可视化

生成：

  • 显著SNP分布图；
  • 显著SNP密度图；
  • 热点分布柱状图；
    输出文件：*_significant_hotspots.png

🔹 3. 最终可视化与总结（3.2.fixed_visualization.R）

3.2.fixed_visualization.R

#!/usr/bin/env Rscript
cat("=== 修复版完整可视化 ===\n")

library(ggplot2)
library(dplyr)
library(viridis)
library(patchwork)
library(circlize)
library(scales)  # 添加scales包用于rescale函数

    echo "错误: 参数不完整"

GWAS_FILE="${BASE_DIR}/color_GWAS.anno.xls"
GFF_FILE="${BASE_DIR}/GCF_902806645.1_cgigas_uk_roslin_v1_genomic.gff.gz"

echo "分析区域: $CHROM:$REGION_START-$REGION_END"

    echo "解压GFF文件..."

    GFF_FILE_PATH="genomic.gff"

$1 == chrom && $2 >= start && $2 <= end && $5 < threshold {

    bcftools view -T "${REGION_NAME}_snps.bed" "$VCF_FILE" -Ov -o "${REGION_NAME}_significant.vcf"

            geom_point(aes(color = ANNO), size = 3, alpha = 0.7) +
            geom_hline(yintercept = -log10(significance_threshold), color = "red", linetype = "dashed", linewidth = 1) +
            scale_color_brewer(palette = "Set1", name = "注释类型") +
            labs(title = paste("GWAS信号 -", region_name, "(P <", format(significance_threshold, scientific=TRUE), ")"),
                 x = "", y = "-log10(P值)") +
            theme_minimal() +
            theme(axis.text.x = element_blank())
        
        # 读取LD数据
        ld_file <- paste0(region_name, "_plink_ld.ld")
        if(file.exists(ld_file)) {
            ld_data <- read.table(ld_file, header = TRUE)
            
            p2 <- ggplot(ld_data, aes(x = BP_A/1e6, y = BP_B/1e6, color = R2)) +
                geom_point(size = 2, alpha = 0.7) +
                scale_color_gradient2(name = "R²", low = "blue", mid = "yellow", high = "red", midpoint = 0.5, limits = c(0, 1)) +
                labs(x = "基因组位置 (Mb)", y = "基因组位置 (Mb)") +
                theme_minimal()
        } else {
            p2 <- ggplot() + 
                geom_text(aes(x = 0.5, y = 0.5, label = "无LD数据")) +
                labs(x = "基因组位置 (Mb)", y = "基因组位置 (Mb)") +
                theme_minimal()
        }
        
        # 组合图形
        combined_plot <- p1 / p2 + 
            plot_layout(heights = c(1, 1))
        
        ggsave("R_LocusZoom_Style.png", combined_plot, width = 10, height = 8, dpi = 300)
        cat("✓ R版LocusZoom风格图生成完成: R_LocusZoom_Style.png\n")
    } else {
        cat("错误: 在指定区域内未找到GWAS数据\n")
    }
} else {
    cat("错误: 未找到GWAS文件\n")
}
EOF2

Rscript generate_r_locuszoom.R "$REGION_NAME" "$REGION_START" "$REGION_END" "$SIGNIFICANCE_THRESHOLD"

echo "=== 阶段6: 清理和总结 ==="

# 清理临时文件
echo "清理临时文件..."
rm -f original_samples.txt new_samples.txt chr_map.txt \
      original_samples_all.txt new_samples_all.txt \

      "${REGION_NAME}_numeric.vcf.gz" "${REGION_NAME}_numeric.vcf.gz.tbi" \
      "${REGION_NAME}_snps.bed" gwas_significant.pvalue genomic.gff \
      generate_high_quality_locuszoom.R generate_r_locuszoom.R \
      "${REGION_NAME}_all_snps.vcf.gz" "${REGION_NAME}_all_snps.vcf.gz.tbi" \
      "${REGION_NAME}_all_snps_fixed.vcf.gz" "${REGION_NAME}_all_snps_fixed.vcf.gz.tbi" \
      "${REGION_NAME}_LD.png" "${REGION_NAME}_LD_simple.png" 2>/dev/null || true

echo ""
echo "=== 综合分析完成! ==="
echo "📁 工作目录: $(pwd)"
echo ""
echo "📊 生成的所有图片文件:"
echo "=== 高质量LocusZoom图 (R版本) ==="
for file in High_Quality_LocusZoom_*.png; do
    if [ -f "$file" ]; then

    fi
done

echo ""
echo "=== 脚本4风格完整图 ==="
for file in Script4_Complete_*.png; do
    if [ -f "$file" ]; then
        size_kb=$(du -k "$file" | cut -f1)
        echo "   📄 $file (${size_kb} KB) - 推荐使用"
    fi
done

echo ""

for file in R_LocusZoom_Style.png; do
    if [ -f "$file" ]; then
        size_kb=$(du -k "$file" | cut -f1)
        echo "   📄 $file (${size_kb} KB)"
    fi
done

echo ""
echo "🔍 分析摘要:"
echo "   显著SNP: $SNP_COUNT 个 (P < $SIGNIFICANCE_THRESHOLD)"
echo "   成功提取: $EXTRACTED_COUNT 个SNP"
echo "   区域SNP: $FINAL_SNP_COUNT 个 (用于LDBlockShow)"

echo "     - 显著性阈值: $SIGNIFICANCE_THRESHOLD"
echo "     - MAF过滤: $MAF_FILTER"
echo "   推荐图片:"
echo "     - Script4_Complete_${REGION_NAME}.png (脚本4风格，一气呵成)"
echo "     - High_Quality_LocusZoom_${REGION_NAME}.png (高质量R版本)"
echo ""
echo "✅ 所有图片生成完成!"
# 给脚本执行权限
chmod +x 4.auto_gwas_hotspot_ld.sh

# 用法1: 使用默认VCF文件分析热点区域
./4.auto_gwas_hotspot_ld.sh "hotspot_31.5_32.0Mb" "NC_047567.1" 31500000 32000000

📚 七、总结

本流程将传统 GWAS 结果从“显著性点的罗列”提升为“聚集信号的结构化分析”，实现了：

  • 🔹 从单点显著到区域热点的层级聚焦；
  • 🔹 自动化 R + Shell 流程，避免人工重复操作；
  • 🔹 为 LD block、候选基因挖掘提供清晰的定位依据。

这套框架可无缝迁移至任意物种或 GWAS 项目，只需更换输入文件与染色体名称即可快速运行。
